body 태그? 

body 태그 안쪽에 본문에 해당하는 내용이 들어간다.

ol ul 등의 태그는 body 안에 들어가있어야 한다

이는 약속이다.

head 태그?

head 태그는 문서를 설명하는 정보를 감싸고 있는 태그다.

즉, 화면에 표시되는 내용들을 body 태그에 넣고, head 태그에는 문서 자체에 대한 정보를 담고 있다.

head 태그에 들어갈 수 있는 또다른 태그로 <title>이 있다.

title 태그로 웹 페이지의 제목을 명명할 수 있다

head 태그와 body 태그를 감쌀 수 있는 상취 태그로 html 태그가 있다.

html 태그는 웹 브라우저에 해당 문서가 html 문서임을 알려주는 역할을 한다.

HTML 표준?

예를 들어 여러 HTML 태그의 필요/불필요 해짐에 따라 태그를 추가/삭제하는 행위

<!DOCTYPE html> 은 무엇을 의미하나?

독타입

이 HTML 문서가 어떤 표준안에 따라서 작성된 것인지를 웹브라우저가 이해할 수 있도록 이 HTML코드를 작성하는 사람이 이렇게 기술해주도록 되어있는 것

작성한 코드가 정확하게 동작하기 위해서는, 코드가 어떤 HTML 표준에 따라 작성되어 있는지 명시해주는 것이 좋다.

각 표준안에 따라서 HTML 태그들이 달라진다.

이것으로 HTML 문법 80% 정도 알게 된 것이다.

HTML 태그는 100 ~ 200개 사이 정도다

영어로 치면 HTML 태그는 영단어.

그렇다고 HTML 태그를 억지로 외울 필요는 없다.
