기획

구상과 기획은 밀접하게 관계 되어있다.

기획 단계에서 많이하는 것 -> UI 모델링

UI 모델링?

UI; User Interface

User: 사용자

Interface: 접점

UI: 사용자가 시스템을 제어하는 조작장치, 우리가 만들 웹사이트 버튼, 화면에 띄우는 정보 등등

모델링: 실제 건물을 짓기 앞서서 모형을 만들어 보는 것, 건축에서 직접 만들기 전에 그것을 가상으로 만들어 보는 것, 이것을 컴퓨터 쪽에서 차용해 사용하는 개념, 사용자의 입장에서 동작하는 기능을 가상으로 만들어 보기

불필요하게 만들고 부시고를 반복하는 것은 여러 자원을 낭비하는 것

무엇을 만들고자 하는지 확실히 공유하는 것이 UI 모델링 이라고 할 수 있다.

웹페이지를 어떻게 배치할 것인지 생각하는 것도 UI 모델링

UI 모델링 도구: 연필로 직접?!, 무료 Pencil(http://pencil.evolus.vn/), 유료 balsamiq(https://balsamiq.com), PowerPoint

기획은 굉장히 중요한것

기획을 꼼꼼하고 사려깊게 하지 못하면 결국에 프로젝트는 반은 실패한 것이라 할 수 있다.
